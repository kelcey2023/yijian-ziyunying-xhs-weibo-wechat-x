# THUQX-智能传播技能套件1.0

THUQX-智能传播技能套件 1.0 是一组基于 OpenClaw 的自动化传播与内容运营 skills，面向多平台内容生成、发布与运营协同场景。

当前仓库收录以下 4 个 skill：

- `zeelin-twitter-web-autopost`：Twitter/X 网页自动发帖、回关与互动
- `zeelin-xiaohongshu-autopost`：小红书内容生成与网页自动发布
- `zeelin-xianyu-auto-ops`：闲鱼消息处理、运营话术与辅助自动化
- `zeelin-report-to-x-autopost`：报告内容到 X 的自动摘要与分发

## 目录结构

```text
.
├── zeelin-twitter-web-autopost/
├── zeelin-xiaohongshu-autopost/
├── zeelin-xianyu-auto-ops/
└── zeelin-report-to-x-autopost/
```

## 使用说明

1. 安装并配置 OpenClaw。
2. 进入对应 skill 目录查看 `SKILL.md`。
3. 按 skill 文档准备浏览器登录状态、脚本依赖与环境参数。
4. 按需在 OpenClaw 中调用对应 skill。
